## ğŸ“‹ Quest System

### Types of Quests

#### 1. **Daily Quests**
- Reset every day
- Must be completed or you'll get penalty quests
- Examples: "Read for 30 minutes", "Exercise for 20 minutes"

#### 2. **Weekly Quests**
- Longer-term objectives
- Reset every week
- Examples: "Complete a project", "Learn a new skill"

#### 3. **Emergency Quests**
- Time-limited urgent missions
- Appear randomly
- High rewards but must be completed quickly

#### 4. **Penalty Quests**
- Automatically generated if you fail daily quests
- Have negative consequences if not completed
- Examples: "Complete 2 extra daily quests", "Lose 50 XP"

#### 5. **Quest Chains**
- Multi-part quests that unlock progressively
- Complete one part to unlock the next
- Higher rewards for completing entire chains

### How to Complete Quests

1. **Accept a Quest**
   - Go to Quests page
   - Click on any quest card
   - Click "Accept Quest"

2. **Do the Real-World Action**
   - The quest tells you what to do
   - Example: "Read Chapter 3 of 'The Art of War'"
   - Go do it in real life!

3. **Report Your Completion**
   - Go to AI tab â†’ Quest Reports tab
   - Select the quest from dropdown
   - Describe what you did, what happened, what you learned
   - Include details: money earned, items acquired, people met, etc.
   - Click "Submit Report"

4. **Get Rewards**
   - System analyzes your report
   - Automatically awards XP, stats, items
   - Shows a Solo Leveling-style reward screen

## ğŸš¨ Troubleshooting

### Loading Screen Stuck
**Problem:** App stuck on "Initializing System..." screen

**Solutions:**
1. Wait up to 10 seconds (safety timeout will hide it)
2. Refresh the page
3. Clear browser cache
4. Check browser console for errors (F12)
5. Try a different browser

### Quest Not Appearing
**Problem:** Quest you expected isn't showing

**Solutions:**
1. Check "Pending Quests" - may be scheduled for later
2. Go to AI tab and share more about your life
3. Complete existing quests first
4. Check if you have too many active quests

### Stats Not Updating
**Problem:** Stats don't change after completing quest

**Solutions:**
1. Make sure you submitted a quest report
2. Check the reward screen - stats should show there
3. Refresh the page
4. Check Stats page - may need to scroll

### Skills Not Working
**Problem:** Active skill doesn't activate

**Solutions:**
1. Check if cooldown is ready
2. Check if you have enough Focus (MP)
3. Make sure skill is unlocked in skill tree
4. Try refreshing the page

### Offline Mode Issues
**Problem:** Features not working offline

**Solutions:**
1. Journal (text) works offline
2. Voice input requires internet
3. Quest generation requires internet (but cached quests work offline)
4. Most features work offline once loaded
